# 競合分析・差別化
> 最終更新: 2026-03-16
## 既存サービスの弱点（バイクツーリング視点）
| サービス | 弱点 |
|---|---|
| Strava | 自転車・ランニング特化。バイクツーリングの「寄り道を楽しむ」文化に合わない |
| Google Maps | ルートを保存できない。ナビ中にルートが勝手に変わる |
| Yahoo!カーナビ | ルートの事前計画機能が貧弱。車向け |
| Garmin/RideWithGPS | 高機能だが複雑。日本の道路事情への最適化が弱い |
## FlexRoute の差別化アイデア
### 1. ルート固定ナビ（★★★ 既に設計済み D-014）
最大の差別化ポイント。ルート計算結果を保存し、ナビ中に再計算しない。
バイクツーリストが自分で選んだ道を走れる。
Google Maps のように「最短ルートに勝手に変更される」不満を解決。
キャッチコピー案：「あなたのルートを、あなたのために。」
### 2. 道路種別の可視化と「道の選択」支援（★★☆）
高速（ピンク）、国道（黄）、県道（緑）、一般道（インディゴ）の色分けは実装済み。拡張案:
- 「高速道路を避ける」「国道を優先する」等のプリファレンス
- ルート全体の道路種別比率をグラフ表示
- バイク乗りの「下道メインで走りたい」ニーズに対応
### 3. ツーリングスポットのコミュニティ共有（★☆☆ Phase 2）
- 保存した「場所」をラベル付きで公開
- 他ユーザーの公開スポットを地図上に表示
- 評価・写真・コメント機能
- 「ツーリングロード」概念（おすすめ道路区間の登録）
### 4. ルートの公開・フォーク（★★☆ Phase 2）
GitHub のようにルートを公開・コピー・カスタマイズ。
- フォークしたルートに寄り道を追加
- 人気ルートランキング（地域別・季節別）
- 派生関係の可視化
- **既存サービスにないユニークな概念**
### 5. 季節・天候・時間帯に基づくルート推薦（★☆☆）
- 走行記録に天候データ（気象庁API）を紐付け
- 「秋の紅葉シーズンに走ったルート」フィルタ
- 日の出・日没時刻を考慮した撮影スポット到着時刻の最適化
### 6. 走破ゲーミフィケーション（★★★ 特許可能性あり）
- 国道コンプリートチャレンジ（国道1号〜507号の走破率）
- 都道府県制覇バッジ（全市区町村通過でバッジ）
- 酷道・険道チャレンジ（国道439号等の走破記録）
- 年間走行距離ランキング（オプトイン）
- **「国道番号ごとの走破率を管理・表示するシステム」は先行技術が見当たらない**
### 7. バイク固有の機能（★★☆）
- グローブモード（大きなタップ領域、手袋操作対応）
- ヘルメット装着時のHUD的表示（情報最小限・大きく）
- エンジン振動を考慮したUI（ボタン大型化、揺れに強いタップ判定）
- 給油アラート（燃費と残燃料から次の給油ポイントを案内）
## Google に依存しないナビアプリのルート計算方式
| 方式 | 代表例 | コスト | 日本語案内 | 道路種別 |
|---|---|---|---|---|
| 自前地図データ + 自前エンジン | ゼンリン（Yahoo!カーナビ）, TomTom, HERE, Garmin | ライセンス料（年間数百万〜） | ◎ | ◎ |
| OSM + オープンソース | OsmAnd, Organic Maps, Magic Earth | サーバー費のみ | △ 英語ベース | △ OSMタグから判定可能 |
| ハイブリッド | Waze, Sygic | 中 | ○ | ○ |
いずれも Google Routes API のような従量課金 API には依存していない。
## Google マイマップからのインポート戦略
### 取得できるもの
- KMLファイル（ポリライン座標、ウェイポイント座標と名前）
### 取得できないもの
- ステップごとの案内文（「国道254号を左折」等）
- 道路種別情報
- 各ステップの距離・所要時間
### 段階的な活用方式
| 方式 | API コスト | ナビ可否 |
|---|---|---|
| A. KMLインポートで閲覧のみ | ゼロ | ✗ |
| B. ナビ開始時のみ Routes API を呼ぶ | 極小（月4回程度） | ◎ |
| C. OSRM でステップを無料取得 | サーバー費のみ | △ 日本語案内なし |
B方式ならルート30本作って実際にナビするのが月4回の場合、Routes API は月4回。
月額 $0.04（約6円）。
