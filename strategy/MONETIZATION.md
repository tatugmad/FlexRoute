# 収益化戦略
> 最終更新: 2026-03-17
## 基本モデル: SaaS + 無料クライアントアプリ
### 構造
```
FlexRoute Web（ブラウザ）
  └─ アカウント登録（無料）
  └─ サブスク課金（Stripe）← 全収益はここで発生
  └─ ルート作成・編集
FlexRoute iOS / Android アプリ（無料）
  └─ ログイン（FlexRoute アカウント）
  └─ ルート閲覧・ナビ実行
  └─ アプリ内課金は提供しない
  └─ 課金ページへのリンクもアプリ内に設置しない
```
### アプリストア手数料の回避
Apple App Store と Google Play の手数料（15〜30%）を回避するため、アプリを「SaaS のクライアント」として位置づける。
| 論点 | 対応 |
|---|---|
| Apple（米国） | 2025年5月 Epic v Apple 判決後、外部決済リンクへの手数料がゼロに。アプリ内に Web 課金リンクを設置可能 |
| Apple（日本含む他地域） | reader アプリ以外は IAP 併用義務があるが、SaaS クライアント（Notion/Slack/Figma 型）は IAP なしで審査通過の前例あり |
| Google Play | アプリ内から外部課金ページへのリンクは禁止。ただし「外部で購入した機能にアクセスするクライアント」は許可 |
FlexRoute は reader アプリ（雑誌・新聞・書籍・オーディオ・音楽・ビデオ）に該当しない。
ただし Notion / Slack / Figma と同じ「SaaS クライアント」として、アプリ内課金なし・外部リンクなしで運用可能。
### 前例（SaaS クライアントモデル）
| サービス | アプリ | 課金場所 | アプリ内 IAP |
|---|---|---|---|
| Notion | 無料 | Web | なし |
| Slack | 無料 | Web / 営業 | なし |
| Figma | 無料 | Web | なし |
| FlexRoute（予定） | 無料 | Web（Stripe） | なし |
### 決済手数料の比較
| 決済方法 | 手数料 | ¥980 の場合の手取り |
|---|---|---|
| Apple IAP（30%） | ¥294 | ¥686 |
| Apple IAP（15%、$1M以下） | ¥147 | ¥833 |
| Google Play（15%） | ¥147 | ¥833 |
| **Stripe（Web 直接）** | **約3.6%** | **約¥945** |
Stripe の Web 直接課金なら、手取りが Apple IAP の1.4倍になる。

## Free プランの API コスト抑制設計
Free プランでは API コールがほぼ発生しない設計が重要:
- ルート3本まで → Routes API 呼び出しを最小化
- ナビなし → Roads API ゼロ
- Nearby Search は Rider 以上に限定
- サムネイルはキャッシュ済みURLを使用（ルート変更時のみ再生成）
